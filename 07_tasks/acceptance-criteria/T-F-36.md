# T-F-36 pyright reportUnknown* を個別 ignore に置換 (真の strict)

## Purpose

pyproject.toml の pyright config から global suppression を全削除し、selected-stack の
`pyright strict` を真に適用する。外部 SDK 由来の type unknown は該当行に
`# pyright: ignore[reportXxx]` を narrow に付与して対処。

## Behavior

### pyproject.toml
- `reportUnknownArgumentType = false` 削除
- `reportUnknownParameterType = false` 削除
- `reportUnknownMemberType = false` 削除
- `reportUnknownVariableType = false` 削除
- `reportUntypedFunctionDecorator = false` 削除
- `reportUntypedBaseClass = false` 削除
- `reportMissingTypeStubs = false` のみ維持 (anthropic SDK 等の stub 未提供を許容)

### 該当ファイル
- `apps/api/src/llm/anthropic.py`: SDK 由来の Any を `# pyright: ignore[...]` で narrow 抑制
- `apps/api/src/db/session.py`: pydantic-settings の generic を narrow 抑制
- (その他 strict 化で出た error は該当行に narrow ignore)

## Acceptance

3-tier AC 全 PASS。Gate #12 selected-stack coverage check rule
(`type_checker_backend = pyright strict`) も PASS。
