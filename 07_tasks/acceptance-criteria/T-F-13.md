# T-F-13 LangGraph + Inngest 基盤

## Purpose

`selected-stack.json#orchestration = "LangGraph + Inngest"` に従い、
LLM ワークフローオーケストレーション基盤を配線する:
- **LangGraph**: 人間承認・スコアループ・状態機械
- **Inngest**: cron / サーバレス job 駆動 (T-F-20 で本格 cron 配線、本 task は import 配線)

## Behavior

### apps/api/src/graph/__init__.py
LangGraph workflow / Inngest client を公開する集約点。

### apps/api/src/graph/workflows.py
LangGraph StateGraph 基盤クラス:
- `AtelierWorkflow` (StateGraph wrapper)
- `WorkflowState` (TypedDict / Pydantic 型)
- node / edge を組み立てる builder helper

### apps/api/inngest_config.py
Inngest client の初期化:
- `inngest_client = inngest.Inngest(app_id="atelier", ...)`
- Inngest function 登録の起点
