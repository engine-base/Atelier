# T-F-31 CI Gate #12 — selected-stack カバレッジ検証

## Purpose

`03_architecture/selected-stack.json` で確定された技術が実装側で配線されているか
CI で機械的に検証する。

## Implementation

`v3-gate.yml` に `gate-12-selected-stack-coverage` job を追加。
inline python で各 category に対応する check rule を実行。

### Check rule 一覧
| category | rule |
|---|---|
| package_manager_ts | `package.json` に `pnpm` |
| package_manager_py | `[tool.uv` in pyproject.toml or `uv.lock` 存在 |
| linter_backend | `[tool.ruff]` in pyproject.toml or `ruff` in apps/api/pyproject.toml |
| linter_frontend | `.eslintrc.json` or `eslint.config.mjs` or `eslint` in apps/web/package.json |
| type_checker_backend | `typeCheckingMode = "strict"` in pyproject.toml |
| type_checker_frontend | `"strict": true` in tsconfig.base.json |
| test_runner_backend | `pytest` in apps/api/pyproject.toml |
| test_runner_frontend | `vitest` in package.json or apps/web/package.json |
| e2e | `@playwright/test` in e2e/package.json |
| git_hooks | `.husky/pre-commit` 存在 |
| monorepo_tool | `turbo.json` 存在 |
| validation_ts | `zod` in package.json or apps/web/package.json |
| validation_py | `pydantic` in apps/api/pyproject.toml |
| ui_core | `tailwindcss` in apps/web/package.json |
| email | `resend` or `@react-email/components` in packages/email/package.json |
| shellcheck | `shellcheck` in .github/workflows/v3-gate.yml |

### Exempt
- `deferred_decisions[]` に列挙された category は skip

## Acceptance

3-tier AC 全 PASS。本 PR 時点で uv / Husky / shellcheck / ESLint / 全ての
selected-stack 配線が完了しているので Gate #12 自体が PASS する。
