# T-A-49 — 受入基準

スキル管理API(F-007)：SKILL.md upload/edit/新規/版/有効化 + AI社員装着

## Tier 1 構造
- UBIQUITOUS: skills の content_md/assets/version/is_active/allowed_employee_roles を CRUD できる API を openapi 通りに公開する
- UBIQUITOUS: スキル管理は管理者(admin)専用でユーザーには非表示

## Tier 2 機能
- [EVENT-DRIVEN] When an admin uploads a SKILL.md, the system shall store content_md and version it
- [EVENT-DRIVEN] When an admin attaches a skill to an AI employee, the system shall update ai_employees.attached_skills
- [UNWANTED] If a non-admin calls skill management endpoints, the system shall return 403 Forbidden （致命）
- [UNWANTED] If the request is unauthenticated, the system shall return 401 Unauthorized （致命）

## Tier 3 回帰
- 既存の ai_employees CRUD が attached_skills 更新後も従来どおり動作する
