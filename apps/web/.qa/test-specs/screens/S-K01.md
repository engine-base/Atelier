### 検出・修正した実バグ

1. GET /knowledge (parent 無指定) が子ノードも返すため、**ツリーのルートに子が重複表示** → `!parent_id` フィルタで是正
2. **パネル折り畳み時に main が 0 幅列に落ちツールバー操作不能** → モック方式 (DOM 維持 + overflow hidden) に修正
3. **TreeNode に循環ガードが無く、自己参照データで無限再帰** (テストで OOM 検出) → parent_id 整合検証を追加
4. ワークスペース未選択だと恒久的に空表示 → 先頭 WS 自動フォールバック (シェルと同じ解決)
