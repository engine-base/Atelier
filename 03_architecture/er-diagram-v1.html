設計メモ
  • UUID v7: 時系列ソート可、insert 順保証、URL 露出安全。uuid_generate_v7() 関数を migration で作成
  • ソフトデリート: 全テーブル deleted_at TIMESTAMPTZ、30日後に cron_schedules → Inngest で hard delete
  • RLS: 全テーブル ENABLE ROW LEVEL SECURITY、Gate #4 で網羅性検証。クライアント招待は invitation_token_hash claim 経由で完全分離
  • ベクトル次元: Voyage AI voyage-3-large は 1024 次元。将来モデル変更時は別カラム追加 + 移行
  • tsvector 自動更新: trigger で content_tsv 列を自動メンテ
  • JSONB 列: settings / tool_calls / items 等。必要時に GIN jsonb_path_ops 追加
  • 暗号化: byok_api_keys.encrypted_key は Supabase Vault で AES-256、復号は service_role のみ
  • マイグレーション: supabase/migrations/YYYYMMDDHHMMSS_<name>.sql、down SQL を併記
  • Generated types: supabase gen types typescript → packages/shared/db-types/、Drizzle schema と並行管理
